# TouchFish - 上班摸鱼，我为自己代言

</div>

> [!Important]
>
>  记住这个插件：`TouchFish` - 让工作时间不再枯燥，摸鱼摸得理直气壮！

一个专为打工人设计的摸鱼神器，让你在 VS Code 里一边看代码，一边看新闻、刷微博，无缝切换，告别无聊。

## 特性

- **多平台新闻聚合**:
  - IT之家
  - 快科技
  - Chiphell
  - V2EX
  - 虎扑
  - NGA
- **全功能微博体验**:
  - **沉浸式阅读**: 支持长微博、图片懒加载，体验飞一般的感觉。
  - **互动达人**: 查看评论、发布微博、关注博主，一个都不能少。
  - **精准筛选**: 支持查看全部、最新、特别关注、好友、热门等微博分类。
- **沉浸式主题**: 深度集成 VS Code 主题，摸鱼不留痕。

> [!Note]
> 老板来了？`Ctrl+Tab` 一秒切回代码，深藏功与名。
